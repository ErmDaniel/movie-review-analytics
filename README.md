# Система рецензирования и анализа фильмов

Python-приложение для хранения, упорядочивания и анализа отзывов и оценок фильмов с поддержкой визуализации данных.

## 🛠 Технологии

- **Python** 3.13.5
- [psycopg2](https://www.psycopg.org/)
- ... (Будет обновляться)

## Итерация 1

Цель первой итерации — реализовать базовую модель данных и минимально работоспособную версию приложения.

- Реализация основных сущностей проекта: Movie, Review, User, а также логики взаимодействия между ними.
- Добавление, редактирование и удаление фильмов, пользовательских оценок и рецензий.

## Итерация 2

Цель второй итерации — расширить функциональность приложения и добавить работу с базой данных и аналитикой.

- Реализация поиска, сортировки и фильтрации фильмов по названию, жанру, году выпуска и средней оценке.
- Интеграция базы данных PostgreSQL и подключение к ней с помощью библиотеки psycopg2
- Реализация слоя хранения данных: сохранение и загрузка информации о фильмах, пользователях, оценках и рецензиях.
- Добавление аналитического модуля на основе Pandas и Matplotlib для построения отчётов и визуализаций.
